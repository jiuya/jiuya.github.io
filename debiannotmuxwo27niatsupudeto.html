debianのtmuxを2.7にアップデート

動機

  • tmux-plugins/tmux-copycatが使いたいがtmux2.3だと動かなかったのでアップデートする。

環境

  • Debian 9.4 stable

インストール

  • Debian stableのパッケージはtmux2.3なのでtestingからパッケージを取ってくる。
  • /etc/apt/sources.list にtesting パッケージを追加する。
# testing
deb http://debian-mirror.sakura.ne.jp/debian/ testing main contrib non-free
deb-src http://debian-mirror.sakura.ne.jp/debian/ testing main contrib non-free
  • インストールする。
$ sudo apt update
$ sudo apt install tmux/testing
  • libc6 のバージョンが低いと怒られるのでlibc6をアップデートする。
  • そのままでは入らないので、依存関係を楽に解決するためにaptitudeを使用する。
  • 良さげな解決方法を採用する。
$ sudo aptitude install libc6-dev/testing libc6/testing
  • その後もう一回やると入る。
$ sudo apt install tmux/testing
$ tmux -V
tmux 2.7

雑感

  • stableでlibc6上げるようなことするなら最初からtesting使っていれば良かったのでは？
  • Desktop環境で次使うときはtestingにしたい。（Debian 10 busterがどうなるかによるけど）
